# Pulsa-App ASCII Football Game

This is a fully playable ASCII football game for Termux (Android) or Linux terminals.

## How to play
1. Clone the repository:
   ```bash
   git clone https://github.com/mjnr0477/Pulsa-app.git
   cd Pulsa-app
   ```
2. Run the game:
   ```bash
   python main.py
   ```

### Controls
- Player1: w/a/s/d, kick e  
- Player2: i/j/k/l, kick o

## Features
- ASCII graphics football field  
- Ball physics  
- Smarter AI opponent  
- Dynamic coach instructions  
- Two human players supported
